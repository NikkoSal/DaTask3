##  Мини-продукт с LLM-аналитикой

Приложение построено по агентной архитектуре:

1. **Пользовательский ввод**
   - Загрузка датасета (CSV / Excel)
   - Ввод запроса и дополнительного контекста

2. **LLM-агент**
   - Используется `create_pandas_dataframe_agent`
   - Модель получает доступ к DataFrame
   - Выполняет Python-код через `PythonAstREPLTool`

3. **Execution Layer**
   - Агент генерирует Python-код
   - Код исполняется в изолированной среде
   - Возвращаются результаты вычислений

4. **Визуализация**
   - Автоматическая генерация графиков 
   - Отображение через Streamlit

5. **UI слой**
   - Streamlit интерфейс
   - Вкладки: данные, графики, агент, история

## Агент

- LLM пишет Python-код
- Код выполняется через `PythonAstREPLTool`
- Результаты вычисляются на реальных данных
- Ответ формируется на основе execution output

## Защита от Prompt Injection

В проекте реализована базовая защита от атак через пользовательский ввод.

### Используемые меры:

1. **Жесткий системный префикс**
   - Запрещает игнорировать инструкции
   - Запрещает раскрытие системных данных

2. **Ограничение области данных**
   - Агент работает ТОЛЬКО с DataFrame

3. **Фильтрация задач**
   - Любые нерелевантные запросы отклоняются


## Визуализация данных

Приложение автоматически строит графики:

- Распределения числовых признаков
- Матрица корреляции
- Агрегации категорий
- Временные ряды (если есть дата)

Графики строятся через `matplotlib`.

## стория взаимодействия

Все запросы пользователя сохраняются в `session_state`.

Функционал:
- Просмотр истории запросов
- Сохранение в JSON

```bash
cd DaTask3
```

```bash
pip install -r requirements.txt
```
